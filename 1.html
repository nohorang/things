HTML

Hypertext Markup Language(HTML) is the stand=dard markup language for creating web pages and web applications. Web browsers recive HTML documents from a web sever or from local storang and render them into multimedia web pages. MTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructsm images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create strucctured documents by denoting structural semantics for text such as headings, paragraphs lists, links, quotes and other items. HTML elements a delineated by tags, written using angle brackets.
